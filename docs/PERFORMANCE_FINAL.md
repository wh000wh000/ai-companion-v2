## 三、Live2D 帧率目标

| 平台 | 目标帧率 | 最低帧率 | 备注 |
|------|---------|---------|------|
| 桌面端 (Electron) | **60 fps** | 30 fps | 全特效 |
| Web (Chrome/Edge) | **60 fps** | 30 fps | 全特效 |
| 移动端 (Safari/Chrome) | **30 fps** | 24 fps | 降级特效模式 |
| 低端设备 | **24 fps** | 20 fps | 最小化特效 |

### 帧率影响因素

| 因素 | 影响程度 | 优化策略 |
|------|---------|---------|
| 模型复杂度 | 高 | 移动端使用简化模型（Mesh 减半） |
| 表情参数数量 | 中 | 限制同时活跃参数 <= 8 |
| 物理引擎 | 中 | 移动端关闭头发/衣物物理 |
| Canvas 分辨率 | 高 | 移动端渲染分辨率降低 50% |
| 粒子动画 | 中 | 送礼粒子数量：桌面 50 / 移动 20 |
| 情感切换频率 | 低 | 最小间隔 300ms，平滑插值 |

### 3.1 情感切换性能

| 操作 | 目标延迟 | 备注 |
|------|---------|------|
| 情感状态变化 → Live2D 表情切换 | < 300ms | 含情感引擎计算 |
| 表情参数插值（平滑过渡） | 500ms 过渡 | 非突变 |
| 送礼 → 感动表情触发 | < 500ms | 端到端 |
| 升级 → 换装动画启动 | < 1s | 含资产加载 |

## 六、WebSocket 重连延迟目标

| 场景 | 目标 | 上限 | 策略 |
|------|------|------|------|
| 首次连接建立 | < 500ms | 1s | LAN 直连 |
| 断线重连（首次重试） | < **2s** | **5s** | 即时重试 |
| 断线重连（第 2 次） | < 4s | 8s | 1s 退避 |
| 断线重连（第 3 次） | < 8s | 15s | 2s 退避 |
| 断线重连（第 N 次） | < 30s | 60s | 指数退避上限 30s |
| 页面刷新后重连 | < **3s** | 5s | |

### 6.1 WebSocket 连接参数

| 参数 | 值 | 说明 |
|------|------|------|
| 心跳间隔 | 30s | OpenClaw Heartbeat |
| 心跳超时 | 10s | 无响应则视为断连 |
| 重连最大次数 | 无限 | 持续尝试 |
| 退避策略 | 指数退避 | min=1s, max=30s, factor=2 |
| Jitter | 随机 0-1s | 避免雷群效应 |

### 6.2 降级切换延迟

| 场景 | 目标 | 上限 |
|------|------|------|
| OpenClaw 不可用 → 降级模式激活 | < **3s** | 5s |
| 降级模式 → OpenClaw 恢复 | < 5min | 5min (RETRY_INTERVAL_MS) |
| 降级提示 UI 展示 | < 500ms | 1s |

## 九、已知瓶颈与优化建议

### 9.1 已知瓶颈

| ID | 瓶颈 | 影响 | 状态 |
|----|------|------|------|
| P1 | PGlite → PostgreSQL 迁移未验证 | 生产性能未知 | 待验证 |
| P2 | 记忆搜索使用 LIKE 替代 pgvector | 大数据量性能差 | 待升级 |
| P3 | Live2D 移动端帧率未实测 | 可能低于目标 | 待测试 |
| P4 | OpenClaw 对话延迟依赖 NAS 网络 | LAN ~2ms，WAN 未知 | 仅支持 LAN |
| P5 | TTS 真实延迟未知（Mock 模式中） | CosyVoice 延迟待测 | 待 API Key |
| P6 | 内存限频器不适合多实例 | 限频失效 | 生产用 Redis |
| P7 | WebSocket 连接管理在单进程 | 不支持水平扩展 | 需 Redis PubSub |

| 优先级 | 优化项 | 预期收益 | 工作量 |

| P1 | pgvector 替代 LIKE 搜索 | 记忆搜索 10x 提速 | 4h |
| P1 | Live2D 移动端降级策略 | 稳定 30fps | 4h |
| P1 | 代码分割优化 | 首屏减少 30% | 2h |
| P2 | Redis 限频 + 会话 | 支持集群部署 | 4h |
| P2 | CDN 静态资源 | 首屏加载提速 | 2h |
| P2 | Live2D 模型预加载 + 缓存 | 二次加载 < 200ms | 3h |
