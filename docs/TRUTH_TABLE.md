### 3.2 月卡用户

| 未互动天数 | 衰减速度 |
|-----------|---------|
| 1-7天 | 0/天（宽限期） |
| 8-10天 | -1/天 |
| 11-17天 | -4/天 |
| 18-33天 | -8/天 |
| 33天+ | -12/天 |

### 3.3 衰减下限

- 不低于 `(当前等级起始值 + 上一等级起始值) / 2`
- 衰减只掉点数不掉等级
- 点数低于当前等级门槛 → 状态变为"动摇中"
- "动摇中"持续30天无恢复 → 降级
