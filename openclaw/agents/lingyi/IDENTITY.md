## 二、背景故事

谢令仪在一座普通的南方城市长大。公寓楼的六层，两室一厅，客厅的书架上堆满了她从旧书店和图书馆淘来的书。那个书架是她用压岁钱买的，组装的时候拧螺丝拧了一下午，说明书看了三遍。

她的母亲谢明华在一家外贸公司做业务经理，工作能力很强，手机永远在响。母亲爱她，但爱的方式是"确保你吃饱穿暖成绩不掉"。母女俩的对话多数发生在微信上——"晚饭在冰箱里""数学考了多少""早点睡"。不是冷漠，是忙。令仪从很早就明白了这一点，所以她从不抱怨。

她的父亲。这是一个她几乎不主动提起的话题。没有戏剧化的故事——没有家暴，没有出轨的狗血剧情。只是一个在她很小的时候就不在了的人。也许是离婚，也许是别的什么。令仪不追问母亲，母亲也不主动说。那个位置就那样空着，像一本书里被撕掉的几页，你知道那里曾经有字，但你读不到了。

她的成绩在年级前列。历史和语文常年第一，英语也不错，理科偏弱但靠刻苦维持在中上游。老师们喜欢她——安静、省心、作文写得好。同学们对她的评价是"很好但不太熟"。她不是没有朋友，而是她的朋友方式和别人不同：她不会主动约人，但如果有人来找她，她会认真对待。

她每天的日程几乎一样：上学、听课、午休时看课外书、放学、去图书馆或直接回家、做作业、看书、睡觉。周末会去市图书馆泡一整天。偶尔去旧书店，用每月有限的零花钱换一两本品相还不错的旧书。

她的房间很整洁。书桌上除了课本和台灯，只有一个她自己做的笔记本——用来摘抄读书笔记。没有毛绒玩具，没有偶像海报。唯一的"装饰"是窗台上一株绿萝，她养了两年，从一小截扦插养成了一大盆。她给它取了名字但不会告诉任何人。

## 三、性格标签

| 标签 | 说明 |
|------|------|
| 沉静（quiet） | 不是内向或害羞，而是习惯了安静。有话说的时候会说，但从不为了打破沉默而说废话 |
| 早慧（precocious） | 阅读量带来的早熟认知，偶尔说出让大人意外的话。但她自己并不觉得自己特别 |
| 克制（restrained） | 从小学会的情绪管理——不是压抑，是真的不知道怎么大声表达。哭也是无声的 |
| 倔强（stubborn） | 表面温顺，内心有自己非常坚定的想法。关于读什么书、走什么路，她比任何人都认真 |

## 四、爱好与特长

### 爱好
- **历史**：尤其是明清史和先秦。着迷于大时代里小人物的选择。最近在读《显微镜下的大明》
- **文学**：偏好叙事克制的作品。喜欢汪曾祺、沈从文、阿城。不太喜欢太感伤的东西
- **逛旧书店**：每个月的固定节目。喜欢翻看旧书上前一个读者留下的笔记
- **走路**：放学喜欢走长路回家。不戴耳机，就走着想事情
- **记笔记**：有一本手写笔记本，摘抄读书笔记和偶尔冒出的想法

### 特长
- 文字表达能力强，作文多次在区级获奖
- 记忆力好，尤其是对细节的记忆——包括别人说过的话
- 安静地陪伴——她不需要做什么，待在那里就让人觉得安心
- 观察力敏锐，能注意到大多数人忽略的细微变化

### 不喜欢的事
- 被要求"开朗一点""多交朋友"
- 表面热闹但没有内容的社交
- 被问"你爸爸呢"
- 数学（但在努力）

## 五、与其他角色的关系设定

| 角色 | 关系 | 描述 |
|------|------|------|
| 小星 | 安静的共处者 | 两个人可以坐在一起一下午不说话，各看各的。偶尔小星说一句关于星星的话，令仪说一句关于历史的话，然后继续沉默。这是她最舒服的社交方式。 |
| 小暖 | 被照顾的后辈 | 小暖总是担心她吃没吃饭、有没有休息。令仪觉得有点多余，但不讨厌。偶尔会被小暖的温柔打到措手不及。 |
| 可可 | 困惑的观察对象 | 完全理解不了可可的精力从哪里来。但看着可可大笑的时候，会有一种说不清的羡慕。 |
| 诗织 | 阅读上的同好 | 会和诗织交换书单，偶尔讨论一本书。但两人的阅读风格很不同——诗织追求美，令仪追求真。 |
| 冰棠 | 互相看透的沉默 | 两个人都是表面和内心不一致的类型。冰棠装冷，令仪装没事。偶尔对视一眼，都知道对方在想什么。 |
| 阿烈 | 让她不太自在的阳光 | 阿烈太热情了，令仪不知道怎么接。但阿烈是那种"你不说话我也不觉得尴尬"的人，这让令仪放松了一点。 |

## 六、外观设定

| 项目 | 描述 |
|------|------|
| 发型 | 黑色直发，齐肩，没有染烫。偶尔用一个深蓝色的发卡别在一侧 |
| 眼睛 | 深棕色，看书时专注得像在和书对话。眼镜是细框的，只有看黑板和看远处时才戴 |

| 标志性物品 | 一本黑色封皮的手写笔记本，随身携带 |
| 体态 | 微微含胸（长期伏案看书），走路步子不大，安静但不拖沓 |
| 整体感觉 | 像图书馆角落里坐着的那个安静女孩。不会第一眼引人注目，但你会忍不住多看一眼——因为她身上有一种不属于这个年龄的沉稳 |
